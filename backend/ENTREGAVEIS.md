## 🚀 PRÓXIMOS 5 MINUTOS

### ⏱️ Minuto 1: Criar Conta Supabase

```
1. Abra https://supabase.com
2. Clique "Start your project"
3. Crie conta (email ou GitHub)
```

### ⏱️ Minuto 2: Criar Projeto

```
1. Clique "New Project"
2. Escolha região
3. Aguarde criação
```

### ⏱️ Minuto 3: Copiar Credenciais

```
1. Project Settings > Database
2. Copie Connection String
3. postgresql://user:pass@host:5432/db
```

### ⏱️ Minuto 4: Configurar .env

```bash
cd backend
cp .env.example .env
# Edite .env com seus valores
DB_USER=postgres
DB_HOST=seu-projeto.supabase.co
DB_PORT=5432
DB_NAME=postgres
DB_PASSWORD=sua-senha
```

### ⏱️ Minuto 5: Testar

```bash
npm install
npm run test:db

# Resultado esperado:
# ✅ TODOS OS TESTES PASSARAM COM SUCESSO!
```
